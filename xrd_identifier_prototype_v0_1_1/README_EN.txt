XRD Identifier Prototype v0.1.1
================================

This is a mojibake-reduced package.

How to build on Windows:
1. Extract this ZIP.
2. Double-click build_exe_windows.bat.
3. The EXE will be created at dist/XRDIdentifierPrototype.exe.
4. Launch the EXE and select a measurement file, reference folder, and output folder.

Sample files:
- sample_data/007_Ba3Cu2Fe24O41_1050C.TXT
- reference_db/private_import/M-type/M-type BaFe12O19 PDF 00-039-1433.txt

Notes:
- This is not Rietveld refinement.
- This tool provides phase candidates, supporting peaks, unmatched peaks, and an identification plot.
- Only M-type reference data is bundled for this prototype.
